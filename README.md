# gfw2pack

A tool that covert [https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt](https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt) to a pac file.

## Usage

```bash
node lib/index.js
```

## Development

Development/dev
```bash
tnpm i
npm run dev
```

build
```bash
npm run build
```

Lint
```bash
npm run lint
```
